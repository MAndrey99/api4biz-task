API
-------
-------

#### /company/add
Добавляет компанию  
Обязательные параметры:  
 - name - название компании

-----

#### /company/list
Возвращает список всех компаний

-----

#### /staff/add
Добавляет работника  
Обязательные параметры:  
  - name     - имя работника  
  - company  - название компании(она должна уже существовать)

-----

#### /staff/list
Возвращает список персонала

-----
